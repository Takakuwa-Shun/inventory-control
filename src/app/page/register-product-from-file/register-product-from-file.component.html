csvファイルから、商品の一括登録が可能です。

登録の際には、データフォーマットに従い、2行目以降から以下の順で記入して下さい。

得意先コード, 得意先名, 商品名, 商品名かな, ボトルコード, ボトル名, トリガーコード, トリガー名, ラベルコード, ラベル名, 詰め替え袋コード, 詰め替え袋名, 内側カートンコード, 内側カートン名, 外側カートンコード, 外側カートン名

  1. csvの文字コードとしてはUTF-8を使用して下さい。（ExcelではShift-jsで保存されることがあります。）
  2. 商品名, 商品名かな以外に関しては、なければ空欄でも構いません。但し、コードと名前がセットでないと登録はされません。
  3. 画像に関しては、一括で登録できないので、画像に関しては個別で後に更新して下さい。
